简单了解一下

首先看一下图.

1.Remote : 远程仓库.就是github主页上显示的仓库.
2.Repository : 本地仓库.先把文件放到本地仓库，再可以放到远程仓库.
3.workspace : 工作区.就是本地的文件夹.(.git文件夹同级目录的地方，就是工作区.)
