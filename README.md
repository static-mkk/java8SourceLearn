# java8源码学习

---
> 主要是通过讲解某个`对象实例`具体的`方法`，来深入了解该类的原理.

因为，如果通篇讲把所有的源码解读，实在是一件非常耗时，并且不易理解.

所以我决定通过讲解常用方法的具体实现，来学习java8的源码.

`好处是 `：

    1. 熟悉常用api
    2. 详细了解某个方法的具体实现
    3. 详细了解该类的原理.
    4. 容易理解.

前面有些使用 `.java` 文件写的，后续会改成 `markdown` 语法的文本文件,这样看起来会比较清晰.
